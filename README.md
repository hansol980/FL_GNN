# 연합학습(Federated Learning) GNN 모델의 DLG 취약점 분석 및 보완 프로젝트

본 프로젝트는 연합학습 환경에서 Graph Neural Network(GNN) 모델이 가질 수 있는 DLG(Deep Leakage from Gradients) 취약점을 분석하고, 이를 방어하기 위한 기반 연구를 진행하는 것을 목표로 한다. 

초기에는 `simpleGCN`을 구현하고 가중치 변화량(Gradient)을 통해 원본 데이터를 유추하는 iDLG 공격을 구현하였다. 그러나 공격 과정에서 가중치의 오차(Gradient Difference)는 0에 가깝게 줄어들지만, 정작 복원된 역산 피처(`dummy_x`)가 원본 노드의 특성과는 전혀 다른 형태로 도출되는 현상이 발생했다. 이는 모델의 구조적 문제라기보다 GNN 환경에서의 DLG 공격이 갖는 전형적이고 고질적인 한계점으로 파악되었다.

초기 모델이 원본 피처를 올바르게 복원하지 못한 주요 원인은 다음과 같다.

1. **메시지 패싱(Message Passing)으로 인한 정보 희석**: GCN 연산 특성상 타겟 노드의 특성 $X$가 이웃 노드들과 먼저 합쳐진(Aggregation) 뒤 가중치 $W$가 곱해진다. 결과적으로 해커가 획득한 그래디언트에는 타겟 노드뿐만 아니라 이웃 노드의 정보가 섞여 있어(Smoothed), 역산 방정식이 다대일(Many-to-One) 혹은 불량조건(Ill-posed) 문제가 되어버린다.
2. **Cora 데이터셋의 희소성(Sparsity)과 규제 부재**: Cora의 특성 벡터(1433 차원)는 대부분이 0으로 이루어진 Bag-of-Words 형태이다. 반면 초기 공격 코드에서는 `dummy_x`를 제약 없는 연속형 정규분포(`torch.randn`)로 초기화하고 Adam 옵티마이저를 사용했기 때문에, 모델이 이산형 벡터(0과 1)를 찾아갈 이유가 없어 엉뚱한 실수 벡터 최적해에 빠지게 된다.
3. **그래디언트에 대한 미미한 기여도**: 연결된 단선(Degree)이 적은 타겟 노드일 경우, 전체 가중치 그래디언트에 미치는 영향력이 극도로 미미하여 최적화가 조기에 종료된다.
4. **ReLU 활성화 함수에 의한 데드존 (Dead ReLU)**: 활성화 함수로 ReLU를 통과할 때 값이 음수가 되면 기울기가 0이 되어 정보가 영구 소실되며, 이는 역전파 과정을 통한 복원에 큰 장애물이 된다.

## 단계별 실험 과정 및 결과 분석

위의 분석을 바탕으로 데이터셋과 초기화 옵션, 훈련 상태, 손실 함수 등을 단계적으로 변경해가며 데이터 복원 성능을 검증하는 과정을 거쳤다.

### 1차 실험: `simpleGNN.py`
* **목표**: Cora 데이터셋을 기반으로 iDLG 알고리즘의 한계를 확인한다.
* **적용 기법**: 초기화된 무작위 `dummy_x`와 정규화 없는 단순 그래디언트 차이(MSE) 손실로 공격을 시도했다.
* **실험 결과**: 
  * **훈련 전(Untrained)**: 코사인 유사도 0.999 이상으로 완벽한 역산이 가능함을 확인했다. (훈련되지 않은 랜덤 가중치는 기울기 소실 없이 모든 특성 정보를 그대로 역전파하기 때문)
  * **훈련 후(Trained)**: 모델 훈련에 따른 가중치 최적화, ReLU 데드존 등의 '정보 병목(Information Bottleneck)' 현상으로 인해 **코사인 유사도가 약 0.057**, **특징 복원 MSE가 약 0.006** 등을 기록하며 원본 특성 복원에 처참히 실패했다 (가중치 오차는 줄지만 원본 복원은 안 되는 대표적 현상).

### 2차 실험: `continuousGNN.py`
* **목표**: Cora 데이터셋의 0과 1만 가지는 희소성(Sparsity) 한계를 극복하기 위해, 완벽한 연속형 실수 벡터를 사용하는 데이터셋으로 교체하여 피쳐 복원을 실험한다.
* **적용 기법**:
  1. **데이터셋 교체**: 300차원의 GloVe 임베딩 특성을 지닌 `WikiCS` 데이터셋을 활용했다. 소수점 단위의 풍부한 피쳐 분포를 가져 연속 데이터 타겟 최적화(Continuous Fitting) 실험에 매우 적합하다.
  2. **최적화 방향 수정**: 기존 0~1 강제 매핑(Sigmoid) 및 L1 규제를 제거하고, 벡터 크기가 발산하지 않도록 L2 Penalty 제약을 걸어 안정성을 높였다.
  3. **활성화 함수 교체**: 음수 영역에서 기울기가 소멸하지 않도록 ReLU 대신 `Tanh`를 사용하여 정보 유실을 원천 방지했다.
* **실험 결과**: 
  * 모델 훈련 후(Trained) 공격 시 **코사인 유사도가 크게 뛴 약 0.636 ~ 0.699**, 특징 복원 MSE 약 0.17 수준을 기록하며 크게 향상되었다.
  * Tanh 교체와 밀집 데이터 조합만으로도 이전보다 훨씬 명확하고 정교한 수준으로 원본 Feature의 흐름을 복원할 수 있음이 증명되었다.

### 3차 실험: `AdvancedDLG.py`
* **목표**: 훈련 시 가중치가 고도화되면서 수반되는 최적화 한계(Gradient Vanishing 등)를 뚫어내기 위해 최신 논문 기술(Advanced DLG)을 도입한다.
* **적용 기법**:
  1. **Cosine Similarity 매칭**: 그래디언트의 절대적 크기가 기하급수적으로 작아지는 특징을 회피하여, 크기 자체(MSE)가 아닌 방향성(Angle)을 맞추는 데 집중했다.
  2. **L-BFGS 최적화기 도입**: 1차 근사에 기반한 Adam 대신, 파라미터 궤적을 2차로 근사하여 탐색하는 L-BFGS 최적화기를 적용해 얕은 지역 최적해(Local Minima) 문제를 타파했다.
  3. **다중 무작위 재시작 (Multi-seed Restarts)**: 초기 무작위 값 난수 의존성을 줄이고자, 3번의 재시작 중 비용(Loss)이 가장 낮게 도출된 `dummy_x`를 최종 채택했다.
* **실험 결과**: 
  * 기존 연속형 피쳐 방식에 고급 수학적 최적화 기법들까지 융합하면서 공격 성능이 비약적으로 강화되었다.
  * 결과적으로 연합학습 파라미터 전송 시 수렴된 상태의 모델일지라도 내부의 미세한 그라디언트 차이를 통해 원본 데이터 및 기밀 유출이 가능해질 수 있음을 확인했다.

---

해당 일련의 과정을 통해, 단순 구조의 GNN이 어떻게 iDLG 공격의 위험망에 들어오게 되는지 분석하였으며, 기밀 보호 체계를 뚫기 위해 해커들이 채용하는 구조 변화(연속형 전환, L-BFGS, 코사인 손실평가 등)의 위협 수준을 검증하였다. 향후 본 실험을 기반으로 GNN 연합학습 모델의 취약점을 근본적으로 방어할 수 있는 솔루션(예: DP 메커니즘, 암호화 등)에 도달하는 것이 이번 프로젝트의 최종 목표이다.
